# updo
> 这是一个用Nodejs+Express实现的支持文件上传下载的简单的web应用。

# Function

- dronzone.js实现文件拖拽上传、下载，可自定义传输容量。
- vuejs实现表格双向数据绑定。
- jquery.form.min.js表单插件，升级表单，实现表单提交回调。
- 纯css+jQuery实现一键返回顶部。
- 简单的ajax异步刷新。
